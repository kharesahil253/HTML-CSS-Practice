WHAT IS ON-PAGE SEO?

On-page SEO, also referred to as “on-site SEO,” is the process of making online pages more user- and search engine-friendly. The optimization of title tags, text, internal links, and URLs are typical on-page SEO techniques.

KEY FEATURES OF ON-PAGE SEO

  • Keyword Analysis – keyword analysis is first step for search engine campaigns. The analyzing process for keywords or search phrases which increase your traffic to your website.
  • Photo Optimization – Image optimization is the process of minimizing the file size of your photographs without compromising quality in order to maintain a fast page load time.
  • CSS Optimization – The cascading style sheet optimization has a important role in seo that describe the website.
  • Content Writing – Content writing is a most important part in seo, this is a meaning full and unique writing for your brand or product.
  • Page Titles – Page title is appear in the browser and search engine. It describes for both search engine and user. It is information about that particular page.
  • Page Headers – It’s a heading line or sub heading lines for content of the page.H1,H2,H3 and more heading tags in html.
  • Meta Descriptions – It is a short summary about your page. Image alt-text – it’s describe image for accessibility
  • Structured Mark-up – The markup known as structured data gives search engines instructions on how to interpret and display content.

WHAT IS OFF-PAGE SEO?

Off-age SEO refers to operations carried out away from a website with the goal of improving its search engine rankings.Building backlinks, promoting branded searches,and boosting social media engagement and shares are typical off-page SEO strategies

KEY FEATURES OF OFF PAGE

  • Keyword Analysis – The analyzing process for keywords or search phrases which increase your traffic to your website..
  • Blog Writing –It is information or article that runs individual.
  • Directory Listing – When there is no index file in a particular website directory, the directory contents are shown using the web server function known as directory listing.
  • Google Indexing – In this list website pages to inform crawler to list these pages on google search engine.
  • Google My Business Listing – This service is provide by google , it list local business.
  • Social Media Publishing – Page creating, image posting free and paid advertising through social media.
  • Backlink Creation – Create back links by document submission, image submission,article submission etc. on other website.
  • Other Unopen – Affiliate marketing, email marketing, etc.

WHAT IS TECHNICAL SEO?

The easiest way to approach technical SEO is to break it down into manageable chunks. If you’re anything like me, you prefer to approach difficult tasks piecemeal and with a checklist. Unbelievably, everything we’ve discussed up to this point falls into one of five categories, each of which calls for a different set of concrete recommendations.

KEY FEATURES OF TECHNICAL SEO

  • Specify a Preferred Domain – Domain is most important for your online business choose best domain with product related name.
  • Optimize Robots.txt – It is txt file that contains do follow and no follow instructions for search engine crawlers.
  • Optimize URL Structure – It is a web address called uniform resource locator, optimize by keywords used in particular page.
  • Add Breadcrumb Menus – Breadcrumbs is like navigation on page that disclose the user location.
  • Canonical URLs – It is a duplicate web address .In this search engine merge duplicate urls
  • POptimize 404 Page – It is an error page that means page is not found., that means particular web address is not present in your website..
  • MSite Map Generation & Optimization – It is a xml file that instruct search engine crawler for page indexing.
